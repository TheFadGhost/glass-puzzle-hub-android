# Glass Puzzle Hub v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `v2.0.0` with high-refresh display preference, repaired Shikaku touch controls, puzzle modes, and future-game recommendations.

**Architecture:** Add pure tested selection/interaction helpers below the UI, then wire them into the Compose shell. Keep v2 focused: Shikaku modes are playable, Sudoku variants and three new games are visible roadmap cards until their engines exist.

**Tech Stack:** Kotlin, Jetpack Compose, Android Window display mode APIs, Gradle Kotlin DSL, Kotlin unit tests.
